# MMusic Shop - Музыкальный магазин

Интернет-магазин музыкальных инструментов и оборудования.

## Функционал
- Каталог товаров
- Корзина с возможностью изменения количества
- Регистрация и вход пользователей
- Личный кабинет с историей заказов
- Админ-панель для отслеживания заказов
- Адаптивный дизайн
